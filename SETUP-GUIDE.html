Bhavya Sampada — Website Deployment Guide

Step-by-step instructions to go live on the internet. Estimated time: 45–60 minutes.

How This Works — The Big Picture

Here is exactly what we are setting up:

  • GitHub (free) — stores all your website files, like a safe vault
  • Netlify (free) — hosts your website on the internet, automatically updates when you change anything
  • GoDaddy — you already have this; we will point your domain to Netlify
  • Decap CMS (free) — your admin panel at bhavyasampada.com/admin to add blogs and videos
✅ Total cost: ₹0/month extra All services used are on free tiers. You only pay for your GoDaddy domain renewal (which you already do).
1
Create a GitHub Account & Upload Your Files
GitHub stores your website files — takes about 10 minutes
1
Create a free GitHub account
Go to github.com and click Sign up. Use your email support@bhavyasampada.com. Choose any username — suggestion: bhavyasampada.
2
Create a new repository
After logging in, click the green New button (top-left).
Fill in:
  • Repository name: bhavyasampada-website
  • Visibility: Private (your files stay private)
  • Check ✅ Add a README file
Click Create repository.
3
Upload all your website files
In your new repository, click Add file → Upload files.

Select all files and folders from the bhavyasampada-website folder (the one you downloaded from me). That includes:
  • 📄 index.html
  • 📄 netlify.toml
  • 📄 SETUP-GUIDE.html
  • 📁 admin/ (entire folder)
  • 📁 content/ (entire folder)
⚠️ Important: Make sure to upload the folders WITH their subfolders, not just the files inside. GitHub will ask you to drag-and-drop entire folders — that works perfectly.
Scroll down, click Commit changes.
2
Connect Netlify to GitHub & Deploy
Netlify will host your website — takes about 10 minutes
1
Create a free Netlify account
Go to netlify.com, click Sign up. Choose Sign up with GitHub — this links them automatically.
2
Import your GitHub repository
On the Netlify dashboard, click Add new site → Import an existing project.
Choose GitHub → find and select bhavyasampada-website.

On the build settings page, leave everything as-is (the netlify.toml file handles this automatically).
Click Deploy site.
✅ In about 30 seconds, your site will be live at a temporary Netlify URL like amazing-cupcake-123456.netlify.app. You can test it!
3
Enable Netlify Identity (required for CMS login)
In Netlify, go to Site configuration → Identity.
Click Enable Identity.

Then scroll down to Git Gateway and click Enable Git Gateway.

Finally, go to Registration and set it to Invite only — this ensures only you can log into the CMS.
4
Invite yourself as the CMS admin
In Netlify → Identity → click Invite users.
Enter your email: support@bhavyasampada.com.
You will receive an email — click the link to set your CMS password.
3
Point Your GoDaddy Domain to Netlify
Connect bhavyasampada.com to your new site — takes 5 minutes (DNS update takes up to 24 hrs to propagate)
1
Add your custom domain in Netlify
In Netlify → Domain management → Add a domain.
Type: bhavyasampada.com → click Verify → Add domain.
2
Update DNS records in GoDaddy
Log in to GoDaddy → go to My Products → Domains → bhavyasampada.com → Manage DNS.

You need to update the A record and add a CNAME record:
Type: A | Name: @ | Value: 75.2.60.5 | TTL: 600 Type: CNAME | Name: www | Value: your-netlify-site-name.netlify.app | TTL: 600
⚠️ Replace your-netlify-site-name with your actual Netlify subdomain (e.g., amazing-cupcake-123456).
Click Save. DNS changes take 2–24 hours to propagate worldwide.
3
Enable free SSL certificate (HTTPS)
Once your domain propagates, go back to Netlify → Domain management → HTTPS.
Click Verify DNS configuration → then Provision certificate.
Netlify will install a free SSL certificate automatically. Your site will show the 🔒 padlock in browsers.
✅ Your site is now live at bhavyasampada.com with HTTPS, fully secured and deployed!
4
How to Use Your CMS Every Day
Add blogs, paste YouTube links — content appears on your website automatically

Visit bhavyasampada.com/admin from any browser. Log in with your email and password (the one you set in Phase 2, Step 4).

📝 To publish a new blog post:
  1. Click 📝 Blog Posts in the left sidebar
  2. Click the existing entry, then click the + Add item button
  3. Fill in: ID (e.g. my-blog-title), Title, Date, Category, Short Excerpt, Full Content
  4. Click Publish (top right)
  5. Wait ~30 seconds → refresh your website → your new article appears! ✅
▶️ To add a YouTube video:
  1. Click ▶️ YouTube Videos in the left sidebar
  2. Click the existing entry, then + Add item
  3. Fill in: Title, YouTube Video ID (see below), Date, Category, Description
  4. Click Publish
  5. Wait ~30 seconds → your video appears embedded on the website ✅
How to find the YouTube Video ID:
Open the video on YouTube. The URL looks like:
https://www.youtube.com/watch?v=dQw4w9WgXcQ
The part after ?v= is the ID. In this example: dQw4w9WgXcQ. Paste just that part.
⚙️ To update your stats or contact details:
  1. Click ⚙️ Site Settings in the left sidebar
  2. Update AUM, clients count, phone number, address etc.
  3. Click Publish — done!
Note: Stats on the homepage hero are currently hard-coded for performance. To update them, you'll need to edit index.html directly or ask your developer. This can be made dynamic in a future update.

✅ Go-Live Checklist

Use this to make sure everything is done before announcing your new website:

  • GitHub account created and all files uploaded
  • Netlify account created and site deployed (green ✅ in Netlify dashboard)
  • Netlify Identity enabled with Git Gateway
  • Yourself invited as admin user and password set
  • GoDaddy DNS A record and CNAME updated
  • Domain live at bhavyasampada.com (may take up to 24 hrs)
  • HTTPS certificate active (🔒 padlock in browser)
  • Tested /admin login successfully
  • Published one test blog post and confirmed it appears on website
  • Added one YouTube video and confirmed it appears on website
  • Tested website on mobile (looks good?)
  • Submitted sitemap to Google Search Console (see below)

🔍 Submit to Google (Important for SEO)

After your site goes live, tell Google about it so it appears in search results:

1. Set up Google Search Console
Go to search.google.com/search-console → Add property → enter bhavyasampada.com.
Follow verification steps (choose the HTML tag method — paste a meta tag Netlify can help you add).
2. Submit your sitemap
In Search Console → Sitemaps → enter: bhavyasampada.com/sitemap.xml
(We'll need to create this file — ask your developer or I can generate it for you.)
